4.安装docker

https://docs.docker.com/engine/install/centos/

1.卸载原来的版本

设置开机自启

sudo systemctl enable docker

配置阿里云镜像加速

2.docker安装mysql

sudo docker pull mysql:5.7

切换root用户

注意\前有空格，-d后台运行

run是启动一个容器，容器与容器互相隔离

-v 前面是虚拟机地址，：后面是容器地址 地址映射

docker ps 查看运行的容器

进入容器命令

docker exec -it mysql /bin/bash

查看目录结构

ls -a

查看跟mysql相关的文件在哪

whereis mysql

exit退出容器

mysql配置

退出容器

cd /mydata/mysql/conf

vi my.cnf

重启mysql

docker restart mysql

3.docker安装redis

1.下载

最新redis

docker pull redis

2.创建实例并启动

mkdir -p /mydata/redis/conf

touch /mydata/redis/conf/redis.conf

3.使用redis镜像执行redis-cli命令连接

docker exec -it redis redis-cli

默认redis是在内存中，无法持久化的,需要配置

exit退出redis

cd /mydata/redis/conf

vi redis.conf

i

appendonly yes

esc

:wq

docker restart redis

4.redis.conf可以配置的地方

5.docker里的容器自启动

docker update redis –restart=always

docker update mysql –restart=always

6.相关

linux环境部署的pdf下，使用命令之前都可以

mkdir -p /root/docker/redis/data

mkdir -p /root/docker/redis/conf

touch redis.conf

#bind 127.0.0.1 //允许远程连接

protected-mode no

appendonly yes

docker run -p 6379:6379 –name redis -v /root/docker/redis/data:/data -v /root/docker/redis/conf/redis.conf:/etc/redis/redis.conf -d redis:5 redis-server /etc/redis/redis.conf

docker exec -it redis redis-cli 进入容器

容器内查看ip

cat /etc/hosts

设置密码
